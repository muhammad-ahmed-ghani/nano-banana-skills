# Category 17: Children's Illustration & Educational Content

Templates for children's book illustrations, educational posters, kids' travel journals, storybook pages, learning materials, and youth-oriented visual content.

Draw a winding, zigzagging travel route in the center with arrows and dotted lines connecting [N] locations. The route connects:
- Stop 1: [ATTRACTION 1] — "[fun child-friendly description]"

- Stop 3: [ATTRACTION 3] — "[fun child-friendly description]"

Surrounding Elements (auto-adapt to the city):

## 17.5 Learning Scene Vocabulary Illustration

**When to use:** User wants an illustrated environment where every object is labeled for language learning.

```
Draw a detailed, richly populated [SCENE — e.g., "pet shop" / "kitchen" / "airport" / "school classroom" / "doctor's office" / "toy store" / "farm" / "underwater scene"] with every visible object clearly labeled.

Label format for each object:
- Line 1: [English / Target language] word (large, clear)
- Line 2: IPA phonetic pronunciation / [romanization]
- Line 3: [Native language translation — Chinese / Spanish / French / etc.]

Label design: thin clean leader lines connecting label to object. Labels in [white rounded rectangles / speech-bubble style / simple flat tags]. No crowding — every label readable at a glance.

Scene density: [richly packed with 20–30 labeled objects / moderate 10–15 objects with breathing room / focused on 5–8 key vocabulary items in context]. Character: [include 1–2 friendly characters interacting with objects / no people / a guide character pointing things out]. Style: [clean vector flat design / warm watercolor / cartoon / realistic for older learners]. Format: [A3 landscape / 16:9 / 1:1].
```
