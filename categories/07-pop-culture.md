# Category 7: Pop Culture & Entertainment

Templates for celebrity mashups, movie scene recreations, album/TV/game edits, and entertainment-themed imagery.

---

## 7.1 Movie / Music Video Scene Recreation

**When to use:** User wants to recreate or place themselves inside a famous scene.

```
A [live television camera operator / film crew / documentary photographer] on the set of [MOVIE / MUSIC VIDEO / TV SHOW — e.g., "Britney Spears' '...Baby One More Time' music video" / "The Matrix lobby scene"], capturing [ICONIC MOMENT] through [a broadcast viewfinder / a handheld camera / a film camera eyepiece].

[ARTIST / CHARACTER], wearing [ICONIC OUTFIT — e.g., "the legendary knotted schoolgirl shirt, pleated skirt, and loafers" / "the long black trench coat"], is [frozen mid-choreography / mid-action] under [hot studio lights / dramatic film lighting]. The set is [a meticulously constructed, brightly lit American high school hallway with lockers and a polished floor / a rain-slicked lobby with marble pillars], filled with [backup dancers / extras / supporting cast].

[DIRECTOR / crew] monitors the shot from [video village / behind a monitor], documenting the creation of a defining [pop culture / cinematic / cultural] moment. Shot on [late-90s broadcast television cameras / 35mm film camera / digital cinema camera]. This behind-the-scenes view captures the raw energy and professionalism behind one of the most memorable [music videos / scenes / moments] in history.
```

## 7.2 Celebrity / Famous Person Crossover Mashup

**Abbey Road / iconic photo recreation:**
```
[PERSON 1], [PERSON 2], [PERSON 3], and [PERSON 4] walk across [a pedestrian crossing like in the Beatles' Abbey Road photo / the Hollywood Walk of Fame / a red carpet], exactly replicating the [composition / spacing / mood / era] of the original. [Additional contextual detail about setting]. Photorealistic, 8K, historically accurate styling for each figure.
```

**TV show / fictional context:**
```
TV show "[SHOW NAME]": [PERSON 1] and [PERSON 2] are [INTERACTION — e.g., "arguing in the courtroom" / "sharing a meal" / "in a tense standoff"]. Scene rendered as authentic [show aesthetic — e.g., "daytime talk show set with bright studio lighting" / "gritty cable drama cinematography" / "sitcom three-camera setup"]. On-screen text overlay: "[SHOW TITLE]" in the show's font.
```

**Video game context:**
```
Show [PERSON 1], [PERSON 2], [PERSON 3], [PERSON 4], and [PERSON 5] together as [CHARACTER CLASS — e.g., "policemen in CS2" / "Overwatch heroes" / "Skyrim adventurers" / "GTA characters"]. Render in the [game's authentic visual style / photorealistic in-game cutscene aesthetic]. Each person's face faithfully recreated in the game's art style.
```

## 7.4 Polaroid Storyboard of a Film or Story

**When to use:** User wants a narrative or film retold as a cork board of Polaroid photos.

```
Create an image about "[FILM / NOVEL / STORY TITLE]" retold through a series of Polaroid photos pinned to a cork board. Each photo captures a key [plot moment / character beat / emotional turning point], with [simple handwritten / typewritten] captions below in the Polaroid white border.

Arrange the photos in a loosely [chronological / thematic] path across the board, using [colored strings / red thread / blue yarn] to connect events and characters in the narrative. Light the scene warmly to evoke nostalgia and mystery. Include incidental details — [coffee cup rings, paper clips, handwritten sticky notes, dried flowers, ticket stubs, small sketches] — for authenticity. [4:3 / 16:9] aspect ratio.
```

## 7.5 "Selfie with" a Fictional Character or Celebrity

**When to use:** User wants a believable selfie alongside a character from a movie, show, or game.

```
A hyper-realistic fisheye wide-angle selfie — without any camera or phone visible in the subject's hands. A close-up, distorted group photo featuring [person from uploaded image] taking a selfie with [CHARACTER(S) — e.g., "Iron Man", "Darth Vader", "Mario and Luigi"]. Everyone is making [wild, exaggerated faces / natural candid expressions / in-character poses], squinting slightly from the flash.

Lighting & Texture: Harsh, direct on-camera flash lighting that creates hard shadows behind the subjects. Authentic film grain, slight motion blur on the edges, and chromatic aberration. Looks like a candid, amateur snapshot captured during a chaotic [behind-the-scenes moment / fan encounter / on-set visit]. [35mm fisheye lens, heavy barrel distortion].
```

**Simple variant:**
```
"I'm taking a selfie with [CHARACTER] on the set of [MOVIE / GAME / SHOW NAME].

Keep the person exactly as shown in the reference image with 100% identical facial features, bone structure, skin tone, facial expression, pose, and appearance. 1:1 aspect ratio, 4K detail."
```

## 7.7 Artistic Document / Sketch Reconstruction

**When to use:** User wants to reverse-engineer or reconstruct the design process of a logo, artwork, or object.

**Logo → Sketches:**
```
Create [6 / 8 / 10] different initial sketches that plausibly led to the final logo design shown in the uploaded image. Show the design process: rough thumbnails, concept explorations, refinements, and near-final iterations. Medium: [pencil on graph paper / ink on white sketchbook pages / digital wireframe printouts]. Arranged as a studio working document — scattered, annotated with margin notes, some crossed out, some circled as favorites. Authentic designer process feel.
```

**Math / Problem Solving on Surface:**
```
Solve [this math problem / this equation / this logic puzzle] shown in the uploaded image and write the full, correct solution [on a whiteboard / on a chalkboard / in the notebook / on the back of the paper shown]. Maintain the handwriting style that matches the original [if shown / in a clean academic style]. Show each step clearly. [Final answer circled / boxed].
```
