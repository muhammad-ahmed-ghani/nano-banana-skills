# Category 4: Infographics & Data Visualization

Templates for product infographics, bento grids, educational visuals, scientific charts, and data-driven imagery.

---

## 4.1 Premium Liquid Glass Bento Grid Product Infographic

**When to use:** User wants a high-end product infographic with modular card layout.

```
Input: [PRODUCT NAME]
Language: [English / Spanish / French / etc.]

Create an image of a premium liquid glass Bento grid product infographic with 8 modules (cards 2–8 show text titles only).

1) Product Analysis:
→ Identify product's dominant natural color → "hero color"
→ Identify category: FOOD / MEDICINE / TECH

2) Color Palette (derived from hero color):
→ Product + accents: full saturation hero color
→ Icons, borders: muted hero (30–40% saturation, never black)

3) Visual Style:
→ Hero product: [real photography (authentic, premium) / 3D Glass version]
→ Cards: Apple liquid glass (85–90% transparent) with whisper-thin borders and subtle drop shadow for floating depth, reflecting the background color
→ Background (choose one):
  - Ethereal: product essence, light caustics, abstract glow
  - Macro: product texture close-up, heavily blurred
  - Pattern: product repeated softly at 10–15% opacity
  - Context: relevant environment, blurred + desaturated
→ Subtle motion effect. Asymmetric Bento grid, 16:9 landscape.
→ Hero card: 28–30% | Info modules: 70–72%

4) Module Content (8 Cards):
M1 — Hero: Product as [real photo / 3D glass / stylized] + product name label
M2 — Core Benefits: 4 unique benefits + hero-color icons
M3 — How to Use: 4 usage methods + icons
M4 — Key Metrics: 5 EXACT data points
  FOOD: Calories [X] kcal/100g, Carbs [X]g, Protein [X]g, [Key Vitamin]: [X]mg, [Key Mineral]: [X]mg
  MEDICINE: Active [name], Strength [X]mg, Onset [X]min, Duration [X]hrs, Half-life [X]hrs
  TECH: Chip [model], Battery [X]hrs, Weight [X]g, [Key spec]: [value], Connectivity [protocols]
M5 — Who It's For: 4 recommended groups with ✓ icons | 3 caution groups with ⚠ icons
M6 — Important Notes: 4 precautions + warning icons
M7 — Quick Reference: [Glycemic Index + dietary tags / Side effects + severity / Compatibility + certifications]
M8 — Did You Know: 3 facts (origin, science, global stat) + icons

Output: 1 image, 16:9 landscape, ultra-premium liquid glass infographic.
```

## 4.2 Landmark / Architecture Infographic

**When to use:** User wants a technical + photographic hybrid infographic of a famous building or structure.

```
Create an infographic image of [LANDMARK — e.g., "the Eiffel Tower" / "the Colosseum" / "the Golden Gate Bridge"], combining a real photograph of the landmark with blueprint-style technical annotations and diagrams overlaid on the image. Include the title "[LANDMARK NAME]" in a hand-drawn box in the corner.

Add white chalk-style sketches showing key structural data: important measurements, material quantities, internal diagrams, load-flow arrows, cross-sections, floor plans, and notable architectural or engineering features.

Style: blueprint aesthetic with white line drawings on the photograph. Technical/architectural annotation style. Educational infographic feel. The real environment visible behind the annotations. [16:9 / 4:3] aspect ratio.
```

## 4.3 Scientific / Technical Data Visualization

**When to use:** User wants a scientific or mathematical concept visualized as a high-quality image.

**Scatter plot / dimensionality reduction:**
```
Visualizing the "[CONCEPT — e.g., Latent Space / Gene Expression Clusters / Population Distribution]" as a [t-SNE / UMAP / PCA] scatter plot. Academic publication quality. Axes labeled with clear tick marks. [Categorical color coding with legend / Density heat-map coloring]. Plot title in top-left. Grid lines subtle, not distracting. Clean white background with thin border. Matplotlib/seaborn publication style. No chart junk.
```

**3D topology / loss landscape:**
```
The 3D "[CONCEPT — e.g., Loss Landscape / Potential Energy Surface / Gravitational Well]" topology. Wireframe mesh surface rendered in [cool blue gradient / fiery red-to-yellow / monochrome]. [X, Y, Z axes labeled / floating axis labels]. Dramatic perspective angle showing both valleys and peaks. Dark background with subtle ambient occlusion. Scientific visualization aesthetic.
```

**Engineering cross-section:**
```
[MACHINE / DEVICE / STRUCTURE] internal cross-section diagram. Cutaway view revealing all internal components. Each component labeled with thin annotation lines and a legend. Color-coded by [material type / function / temperature]. Technical illustration style: clean linework, subtle shading, no textures. White background. Scale bar in corner.
```

## 4.5 Facial / Age Analysis Infographic

**When to use:** User wants a futuristic AI analysis report overlaid on a portrait.

```
A hyper-realistic, high-resolution portrait infographic based on the uploaded photo. Keep the same person, identity, hairstyle, clothing, and natural skin tone. Use a neutral studio background.

Overlay a subtle, semi-transparent facial analysis grid on the entire face — very similar to a 3D face-scanning mesh: thin, soft white lines following the facial contours, slightly glowing but not hiding the skin details. Add one clean vertical red laser line running down one side of the face, like a futuristic scan.

Create a clean medical-aesthetic infographic evaluating 5 aging factors:
1. Fine lines and wrinkles — "[X]%"
2. Skin texture and elasticity — "[X]%"
3. Facial volume and sagging — "[X]%"
4. Eye area aging signs — "[X]%"
5. Skin tone and pigmentation — "[X]%"

For each factor: a small label with a thin line pointing to the relevant facial area. Use clean, modern, sans-serif typography and small technical-style text, like a scientific facial analysis UI.

At the bottom center: large bold "ESTIMATED AGE: [X]"

Overall style: futuristic AI-guided skincare analysis, minimalistic, premium editorial lighting. No gender mentioned. Suitable for any human face.
```

## 4.6 Map-Style Visualizations

**When to use:** User wants a location or geographic area rendered in a specific visual style.

**Watercolor map:**
```
Generate a [watercolor / hand-painted / illustrated] map of [REGION — e.g., "Germany" / "Japan" / "New York City"] where all [federal states / prefectures / neighborhoods] are labeled in [ballpoint pen / handwritten ink / printed serif]. Include [rivers, mountain ranges, major cities] as stylized watercolor elements. Color palette: [soft pastels / rich earth tones / blue and green washes]. Border: thin ink outline. Paper texture visible underneath.
```

**8-bit / pixel art map:**
```
[8-bit style / retro pixel art] map of [LOCATION — e.g., "San Francisco" / "Tokyo" / "Paris"]. Authentic retro game color palette. Pixelated terrain features: [water in blue pixels / forests in green pixel clusters / buildings as brown and grey blocks]. Roads as thin pixel lines. Key landmarks identifiable as pixel sprites. [Optional: animated pixel characters walking the streets].
```

**Google Street View style:**
```
Google Street View photograph at [COORDINATES — e.g., "36.4602° N, 25.3730° E"] at [TIME — e.g., "sunset / 14:00 / blue hour"]. Authentic Street View visual style: [fisheye lens distortion / panoramic stitching seams / camera car reflection in windows]. The location and time of day should be geographically and atmospherically accurate. Photorealistic, environmental detail.
```
