# Category 8: Architecture, Maps & Environment

Templates for architectural blueprints, map styles, interior design, isometric diagrams, and location-based imagery.

---

## 8.1 Location-Based Image from Coordinates or Description

**When to use:** User wants an image generated based on a specific geographic location and time.

```
Create an image of [LATITUDE° N/S LONGITUDE° E/W] at [TIME — e.g., "sunset / 19:00 / blue hour / midday"].
```

**Expanded version:**
```
Generate a photorealistic image of the scene at coordinates [LATITUDE, LONGITUDE] at [TIME OF DAY]. The environment should reflect the accurate: geography and terrain of that location, local architecture and urban/rural character, seasonal conditions [for March / summer / winter], atmospheric lighting at [sunrise / golden hour / dusk / night]. Style: [travel photography / street photography / aerial drone shot / Google Street View aesthetic]. 8K, geographically faithful.
```

**Named location:**
```
Google Street View photograph of [LOCATION — e.g., "the Amalfi Coast road, Italy" / "Shibuya Crossing, Tokyo" / "Santorini village path"] at [TIME]. Authentic Street View visual style: fisheye lens, panoramic stitching seams, road-level perspective, camera van reflection in windows. Photorealistic, geographically accurate environment.
```

## 8.3 Isometric / Hand-Drawn Architectural Diagram

**When to use:** User wants a stylized isometric view of a building, district, or object.

**Text → Isometric:**
```
Create a hand-drawn isometric schematic diagram of [SUBJECT — e.g., "a medieval castle" / "a ramen restaurant kitchen" / "an airport terminal"]. Style: architectural sketch — precise ink linework on [clean white / aged cream / graph paper] background. Isometric 3D projection showing [exterior / interior cross-section / both]. Key elements labeled with thin annotation lines in [handwritten / technical stencil] style. Hatching for shadows. Scale bar and North arrow in corner.
```

**Photo → Isometric:**
```
Create a hand-drawn isometric schematic diagram of the [street / building / space / object] shown in the uploaded photo. Accurately capture the spatial relationships, key structural elements, and proportions. Style: clean architectural sketch with ink outlines. [Add dimension annotations / Keep as clean outline only].
```

**Isometric → Photo:**
```
Create a photorealistic photograph from this hand-drawn isometric schematic diagram. Interpret the structural diagram as actual [architecture / product / space] with real materials: [concrete, glass, steel / brick and wood / stone and greenery]. Lighting: [golden hour / overcast soft light / dramatic twilight]. Include environmental context: sky, vegetation, pedestrians for scale. 8K architectural photography.
```

## 8.4 Blueprint Technical Diagram

**When to use:** User wants an engineering-style technical drawing of a product or structure.

```
A [two-panel / single-panel] technical blueprint diagram in clean monochrome line-art.

LEFT PANEL: A full, intact [OBJECT — e.g., "Sony PlayStation 5" / "vintage motorcycle" / "Victorian greenhouse"] in precise thin line-art on a [white / dark navy] background.

RIGHT PANEL: A highly detailed, vertically exploded-view diagram of the [OBJECT] showing each component separated into layers: [list 8–12 components, e.g., "outer shell panels, cooling fan, heatsink, motherboard, SSD module, power supply, internal frame, ports, vents, base stand, screws"].

Use consistent thin [grey / white] line-weight with no shading. Add numbered circular labels around each part, with a matching numbered parts list at the bottom — exactly as in a professional technical manual. Place [BRAND LOGO / OBJECT NAME] text in the top left in the engineering-manual style. Mirror the composition, spacing, and minimalist aesthetic of a real technical schematic.
```

**When to use:** User wants a conceptual or abstract scientific environment rendered visually.

**Magnetic field / physics visualization:**
```
[Tokamak Fusion Reactor Magnetic Field Lines / Solar wind plasma interaction / Gravitational lensing around a black hole / Quantum field fluctuation in vacuum] visualized as a photorealistic scientific render. [Flowing magnetic field lines as luminous blue-white arcs / Plasma in vivid orange-magenta torus / Gravitational distortion as warped star fields]. Dark space background with subtle starfield. Scientific accuracy balanced with visual drama. [Labels for key phenomena in clean sans-serif / No labels, pure visual]. Resolution: 8K, publication quality.
```

**3D topology visualization:**
```
The 3D "[TOPOLOGY NAME — e.g., Loss Landscape / Energy Surface / Potential Well / Riemann surface]" rendered as a dramatic surface topology. Wireframe mesh in [cool blue gradient shading / fiery red-to-yellow heat map / monochrome with edge emphasis]. Perspective angle: [bird's-eye showing full basin structure / dramatic low angle showing scale of peaks and valleys]. [X, Y, Z axes with clean labels / floating axis labels / no labels]. Dark background with ambient occlusion. Mathematical visualization aesthetic — rigorous but visually striking.
```
