# Category 18: Storyboard & Sequential Narrative

Templates for generating professional storyboards, comic panels, shot plans, and sequential narrative grids. Covers all major coverage types used in film, advertising, animation, and social content production.

---

## QUICK REFERENCE: Shot Sizes, Angles & Transitions

**Shot Sizes (smallest → widest):**
- `ECU` — Extreme Close-Up: single detail (eye, lips, logo, hand)
- `CU` — Close-Up: face fills frame (chin to hairline)
- `MCU` — Medium Close-Up: chest to crown ("talking head")
- `MS` — Medium Shot: waist to crown
- `MLS` — Medium Long Shot / Cowboy Shot: mid-thigh to crown
- `LS` — Long Shot / Full Shot: full body + immediate environment
- `WS` — Wide Shot: character small, environment dominant
- `EWS` — Extreme Wide Shot / Establishing Shot: vast scale, tiny or no characters

**Camera Angles:**
- `EL` — Eye Level (neutral, natural)
- `LA` — Low Angle, looking up (power, heroism, menace)
- `HA` — High Angle, looking down (vulnerability, smallness, surveillance)
- `OHD` — Overhead / Bird's Eye (top-down, map-like, omniscient)
- `DA` — Dutch / Canted Angle (unease, disorientation, tension)
- `POV` — Point of View (subjective — what the character sees)
- `OTS` — Over The Shoulder (camera behind character A, facing character B)

**Transitions (write between panels):**
- `CUT` — Instantaneous hard cut
- `DISSOLVE` — Overlap fade between shots
- `FADE TO BLACK` / `FADE IN` — Scene begins or ends
- `MATCH CUT` — Same composition, different location/time
- `JUMP CUT` — Same angle, skip in time (disorienting)
- `SMASH CUT` — Jarring abrupt transition for shock
- `WIPE` — One image pushes the other off frame

## 18.1 Master Storyboard Template (Any Grid)

**When to use:** Universal template — adapt grid, shot plan, and style to any project.

```
Create ONE single-image storyboard sheet with NO additional text outside the image.

NON-NEGOTIABLE LAYOUT CONTRACT:
- Grid: exactly {COLS} columns × {ROWS} rows = {PANELS} panels total.
- Perfect rectilinear layout only — no tilted, curved, or irregular panels.
- All panels exactly equal size with uniform gutters between them.
- Uniform outer margin around the entire grid.
- No merged panels, no split panels, no missing panels, no extra panels.
- No visual elements crossing panel boundaries.
- Reading order: left-to-right, top-to-bottom.
- Consistent character appearance, color palette, and art style across ALL panels.
- The storyboard grid must fill the entire image — not a small portion.

Header (above grid): "{STORY TITLE}" | Scene {SCENE_NUMBER} | Director: {NAME}

Each panel contains:
- The scene as described in the shot plan
- Panel number (small, bottom-left corner of each panel)
- Shot size label (bottom-right corner: ECU / CU / MCU / MS / MLS / LS / WS / EWS)
- Scene/panel description text (below each panel in a clean text strip)
- Camera direction arrows overlaid where movement is specified

Story & Visual Style: {STORY_PROMPT_AND_STYLE}

SHOT PLAN ({PANELS} panels, reading left-to-right top-to-bottom):
{SHOT_PLAN}

Technical: [sketch pencil / clean line art / minimal color / full color / cinematic dark / bright graphic novel]. Aspect ratio: [{16:9 / 4:3 / 9:16} panels within the sheet].
```

## 18.6 CHARACTER_ACTION Coverage

**When to use:** Showing a character performing a physical action across multiple panels — running, fighting, falling, dancing, cooking, building. Focus on spatial continuity and motion progression.

```
Create ONE single-image CHARACTER_ACTION storyboard sheet — a {COLS}×{ROWS} grid = {PANELS} panels.

LAYOUT CONTRACT: [same NON-NEGOTIABLE rules as master template above]

ACTION_COVERAGE RULES:
- Panel 1 MUST be an establishing shot (WS or EWS) showing the full environment and character in it.
- Each subsequent panel must advance the action by one clear beat — no frozen or repeated moments.
- Maintain spatial consistency: character always moves in the same screen direction unless an intentional reversal is shown.
- Show motion blur, action lines, or mid-movement poses — not static standing poses.
- Camera moves closer as tension builds (WS → LS → MS → MCU → CU for the climax beat).
- The character's costume, body proportions, and physical features must be IDENTICAL across all panels.
- The environment orientation, lighting direction, and color palette must be consistent.

Action being storyboarded: {DESCRIBE THE ACTION — e.g., "a parkour runner vaulting over rooftop obstacles across a city skyline at dusk"}

SHOT PLAN ({PANELS} panels):
Panel 1: EWS — Establish environment. Character small in frame, about to begin the action.
Panel 2: WS — Character begins the action. First beat of movement.
Panel 3: MS — Mid-action. Body mechanics of the movement clearly visible.
Panel 4: MCU — Reaction / effort / expression at the peak of the action.
Panel 5: CU — Key detail: {hands gripping / feet landing / face impact}.
Panel 6: WS/LS — Resolution of the action. Character at the end position.
[Adjust panel count and shots to match your grid and action beats.]

Art style: {e.g., "dynamic manga-style action lines, high contrast black and white with speed blur effects" / "clean animation pre-viz — simple shapes, no shading" / "cinematic storyboard pencil sketch"}.
```

## 18.7 DIALOGUE_COVERAGE

**When to use:** A conversation between two or more characters. Classic Hollywood dialogue coverage: establish → OTS shots → singles → reactions.

```
Create ONE single-image DIALOGUE_COVERAGE storyboard sheet — a {COLS}×{ROWS} grid = {PANELS} panels.

LAYOUT CONTRACT: [NON-NEGOTIABLE rules as master template]

DIALOGUE_COVERAGE RULES:
- 180° LINE RULE: Camera must stay on the same side of the imaginary axis between characters throughout. Never cross the 180° line unless a deliberate visual mismatch is intended.
- Standard dialogue coverage sequence: Establish → OTS (Character A) → OTS (Character B) → Single MCU (A) → Single MCU (B) → Reaction CU → Resolution.
- OTS shots: camera positioned BEHIND one character's shoulder, facing the other. The near character's shoulder/ear is visible in foreground.
- Single shots: camera is directly on one character with no other character visible.
- Reactions: CU or MCU, no dialogue — pure listening/reacting expression.
- Dialogue line for each panel written in the caption strip below.
- Each character must be visually distinct and consistent in appearance across ALL panels.

Scene: {DESCRIBE THE CONVERSATION — location, characters, topic, emotional stakes, e.g., "Two detectives in a dimly lit interrogation room. Detective A (stern, late 40s, suit) interrogates Suspect B (nervous, young, hoodie) about a missing artifact."}

SHOT PLAN ({PANELS} panels):
Panel 1: EWS/WS — Establish location. Both characters visible in the space. Set emotional tone.
Panel 2: MS/WS — Master shot. Both characters in frame, their spatial relationship established.
Panel 3: OTS-A→B — Camera behind Character A, facing Character B. Character A speaks: "{DIALOGUE LINE A}"
Panel 4: OTS-B→A — Camera behind Character B, facing Character A. Character B responds: "{DIALOGUE LINE B}"
Panel 5: MCU-A — Single on Character A. Speaking or reacting. "{DIALOGUE/ACTION}"
Panel 6: MCU-B — Single on Character B. Reaction or response. "{DIALOGUE/ACTION}"
Panel 7: CU-A — Reaction close-up. Expression reveals inner state.
Panel 8: CU-B — Counter-reaction. Emotional shift.
Panel 9: WS — Resolution. End of exchange, characters repositioned or one exits.

Art style: {e.g., "realistic noir pencil — heavy shadows, cross-hatching / clean animation pre-viz / graphic novel ink and color"}.
```

## 18.8 SCENE_COVERAGE (Master Coverage)

**When to use:** Full production coverage of a single scene — all the shots a director would need to edit the scene in post. Includes master, complementary angles, inserts, cutaways.

```
Create ONE single-image SCENE_COVERAGE storyboard sheet — a {COLS}×{ROWS} grid = {PANELS} panels.

LAYOUT CONTRACT: [NON-NEGOTIABLE rules as master template]

SCENE_COVERAGE RULES:
- Panel 1 MUST be the Master Shot: a WS or EWS capturing the entire scene and all characters in a single take.
- Subsequent panels provide coverage from different angles of the SAME scene — not plot progression.
- All panels depict the SAME moment/scene from different POVs and sizes.
- Include: [Master Shot / Complementary angles / Reverses / Close-ups / Inserts / Reaction shots].
- The scene itself does not advance — this is pure cinematographic coverage.

Scene being covered: {DESCRIBE THE SCENE — location, all characters, action, e.g., "A woman signs a contract at a large boardroom table while three lawyers watch in silence, late afternoon light through floor-to-ceiling windows"}

SHOT PLAN ({PANELS} panels — all covering the same scene from different angles):
Panel 1: EWS/WS — Master shot. Full scene, all characters visible. Establishes the definitive geography.
Panel 2: MS — Complementary angle. Same scene, camera repositioned 90° from master.
Panel 3: OTS — Over the shoulder. From behind primary character, facing secondary.
Panel 4: OTS — Reverse OTS. Flipped — from behind secondary character, facing primary.
Panel 5: MCU — Single on Character A. Same action, tight.
Panel 6: MCU — Single on Character B. Same action, tight.
Panel 7: CU — Insert: key detail of the action (the signing hand, the contract, the nervous eye twitch).
Panel 8: CU — Reaction: the most important reaction in the scene (who's watching, who's worried).
Panel 9: WS — Release/pull-out. Same master angle, the conclusion of the scene action.

Art style: {e.g., "professional AD sketch — clean pencil with gray wash / TV drama storyboard / full color cinematic"}.
```

- Consistent composition: maintain the same camera position/framing across panels to make the time change the ONLY variable.
- Caption strip below each panel: the time/date this panel represents.

Panel 3: {NEXT TIME} — Caption: "{timestamp/date}" — {describe what's different}
[Continue for all panels...]
Panel {N}: {LATEST TIME} — Caption: "{timestamp/date}" — {final state}

Time indicators in each panel: [{clock on wall / sunrise→noon→sunset light angles / winter snow→spring flowers→summer heat→autumn leaves / character hairstyle + wrinkles aging / newspaper date in background}]

## 18.11 SCENE_JUMP / Cross-Cut Sequence

**When to use:** Parallel editing between two or more simultaneously occurring storylines in different locations (cross-cutting). Classic thriller/suspense/rescue narrative technique.

```
Create ONE single-image SCENE_JUMP storyboard sheet — a {COLS}×{ROWS} grid = {PANELS} panels.

LAYOUT CONTRACT: [NON-NEGOTIABLE rules as master template]

SCENE_JUMP RULES:
- Panels alternate between 2 [or more] distinct, simultaneous storylines in DIFFERENT locations.
- Every time the location changes, a SCENE HEADING label appears at the top of that panel: ["INT. LOCATION A — NIGHT" / "EXT. LOCATION B — SAME"].
- Each location must have a distinct, consistent color palette to help the viewer instantly identify which storyline they're in: [Location A = warm amber tones / Location B = cold blue tones].
- Characters are location-specific — Character A never appears in Location B's panels and vice versa.
- The cutting rhythm between storylines builds tension — later panels alternate faster (shorter story beats per panel).

Storyline A: {DESCRIBE — e.g., "A hostage negotiator on a rooftop, trying to defuse a bomb with 3 minutes left"}
Storyline B: {DESCRIBE — e.g., "The bomb squad racing through city traffic in an armored vehicle, 4 blocks away"}

CROSS-CUT SHOT PLAN ({PANELS} panels):
Panel 1: [STORY A] EWS — Establish Location A. Scene heading top of panel.
Panel 2: [STORY B] EWS — Establish Location B. Scene heading top of panel. Distinct color palette.
Panel 3: [STORY A] MS — Character A — advancing their situation.
Panel 4: [STORY B] MS — Character B — advancing their situation simultaneously.
Panel 5: [STORY A] MCU — Tension beat for Character A.
Panel 6: [STORY B] MCU — Tension beat for Character B.
Panel 7: [STORY A] CU — Character A's critical action / peak tension.
Panel 8: [STORY B] CU — Character B's critical response / peak tension.
Panel 9: [BOTH / CONVERGENCE] WS — The two storylines converge in one frame OR the resolution of one.

Art style: {e.g., "high-tension noir — heavy shadow, split tone / thriller TV drama storyboard / graphic novel with color-coded storylines"}.
```

## 18.12 MONTAGE Sequence

**When to use:** A rapid sequence of thematically linked images designed to show progress, emotion, a journey, or the passage of experience — not a linear story.

```
Create ONE single-image MONTAGE storyboard sheet — a {COLS}×{ROWS} grid = {PANELS} panels.

LAYOUT CONTRACT: [NON-NEGOTIABLE rules as master template]

MONTAGE RULES:
- Each panel is a distinct image — not connected by spatial or dialogue continuity.
- All panels share a unifying theme, subject, or emotional through-line.
- Shot variety is ESSENTIAL: alternate ECU → WS → CU → MS → OHD → LA freely — a montage never stays at the same shot size for two consecutive panels.
- No dialogue, no speech — pure visual language.
- The emotional arc of the montage must read across the panels in sequence: [e.g., struggle → effort → failure → determination → triumph] or [departure → journey → discovery → arrival].
- Transitions between panels may be visualized as [DISSOLVE / FLASH CUT / MATCH CUT] — indicate in caption strip.

Montage theme and subject: {DESCRIBE — e.g., "A training montage showing an underdog boxer going from unfit and losing fights to championship-ready" / "A travel montage of one couple's world trip — airports, temples, markets, mountains, beaches" / "A product launch montage — late nights, prototypes, failures, team huddles, launch day"}

MONTAGE SHOT PLAN ({PANELS} panels):
Panel 1: [OPENING — Establish the starting state / subject / character]
Panel 2: [1st activity / effort / journey moment — wide or environmental]
Panel 3: [Close detail — hands, face, key object in the theme]
Panel 4: [Low point or contrast beat — failure, exhaustion, obstacle]
Panel 5: [Recovery — determination, renewed effort]
Panel 6: [Progress visible — compare to Panel 1]
Panel 7: [Peak effort — the hardest moment, most intense]
Panel 8: [The threshold — about to break through]
Panel 9: [RESOLUTION — Achievement, arrival, transformation. Match composition to Panel 1 to show contrast.]

Caption strip format: [Transition type between panels: CUT / DISSOLVE / MATCH CUT]
Music/rhythm note (optional, in header): {BPM or tempo — e.g., "Fast cut rhythm — 12 frames per panel beat"}

Art style: {e.g., "dynamic sports graphic — high contrast, motion blur / documentary photograph / inspirational warm editorial"}.
```

## 18.14 Comic / Manga Page (Single Story Page)

**When to use:** A sequential art page in comic, graphic novel, or manga style — prioritizes visual storytelling, dynamic panel arrangements, and typographic speech bubbles.

```
Create ONE single-image comic page in [{American comic / manga / Franco-Belgian BD / webcomic} style].

LAYOUT: {Choose one:}
OPTION A — STRICT GRID: {COLS}×{ROWS} = {N} equal panels. [Apply same NON-NEGOTIABLE layout rules.]
OPTION B — DYNAMIC LAYOUT: Panels of varying sizes (a large establishing panel + smaller action panels). No grid required. All panels separated by gutters. No bleed, no merges unless intentional splash panel.

PAGE COMPOSITION (Option B — dynamic):
- One SPLASH PANEL: large, occupying [{top half / full left column / ⅓ of the page}] — the most dramatic moment.
- Supporting panels: {N} smaller panels arranged around the splash.
- All panels rectilinear (straight edges). Gutters consistent throughout.

COMIC_STYLE RULES:
- Speech bubbles: hand-lettered or clean comic font (ALL CAPS inside bubbles). Dialogue exactly as written below. Bubble tails point to speaker accurately.
- Sound effects: bold onomatopoeia integrated into the action panel artwork (BAM, CRACK, THOOM, SHHH, etc.).
- Emotion lines, speed lines, motion blur: as appropriate for the action.
- Panel borders: [black ruled border / borderless / with drop shadow].
- Gutters: [white / black / colored — match genre: black for dark/noir, white for all-ages].

Story page content: {DESCRIBE THE SCENE AND ACTION ON THIS PAGE}

PANEL PLAN:
Panel 1: {scene} — DIALOGUE: "{line}" — SOUND FX: {if any}
Panel 2: {scene} — DIALOGUE: "{line}"
Panel 3: {scene} — SOUND FX: {BAM / CRASH / etc.} — NO DIALOGUE, pure action
Panel 4: {scene} — DIALOGUE: "{reaction line}"
[Continue for all panels...]

Art style specifics: {e.g., "1990s Image Comics — Jim Lee style, heavy crosshatch, dramatic anatomy / Moebius Franco-Belgian — clean line, pastel color / Junji Ito manga — detailed horror, precise black ink / Saturday morning cartoon — bold outlines, bright primaries, all-ages"}.
```
