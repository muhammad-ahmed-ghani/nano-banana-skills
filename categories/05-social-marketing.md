# Category 5: Social Media & Marketing

Templates for quote cards, magazine covers, thumbnails, posters, and content designed for sharing.

---

## 5.1 Wide Quote Card with Portrait

**When to use:** User wants a shareable quote card featuring a person and typography.

```
A wide quote card featuring [a famous person / the person in the uploaded image / an abstract human silhouette], with a [brown / dark navy / deep burgundy / warm cream] background and a [light-gold / white / charcoal] serif font for the quote: "[QUOTE TEXT]" and smaller text: "—[AUTHOR NAME]." There is a large, subtle quotation mark before the text.

The portrait of the person is on the [left / right], the text on the [right / left]. The text occupies [two-thirds] of the image and the portrait [one-third], with a slight gradient transition effect on the portrait.

Typography: [serif / sans-serif], [elegant / bold / handwritten accent]. Layout: [landscape 16:9 / square 1:1 / portrait 4:5]. Overall mood: [inspirational / thoughtful / powerful / nostalgic].
```

**Raycast-parameterized version:**
```
A wide quote card featuring a famous person, with a [brown] background and a light-gold serif font for the quote: "{argument name="famous_quote" default="Stay Hungry, Stay Foolish"}" and smaller text: "—{argument name="author" default="Steve Jobs"}." There is a large, subtle quotation mark before the text. The portrait is on the left, text on the right. Text occupies two-thirds of the image, portrait one-third, with a slight gradient transition on the portrait.
```

## 5.2 Magazine Cover

**When to use:** User wants to create a realistic magazine cover.

```
A photo of a glossy magazine cover. The cover has the large bold words "[MAGAZINE TITLE]". The text is in a [serif / sans-serif] font, [black on white / white on dark background / gold on navy], and fills the view. [No other text / Include 3–4 cover line teasers in smaller text.]

In front of the text there is a dynamic portrait of [a person in [STYLE] fashion / a product shot / a dramatic landscape]. [Color palette: green and banana yellow / bold primary colors / monochrome with one accent color.]

Include [the issue number], [today's date or month/year], [a barcode], and [a price] in the appropriate corners. The magazine is [on a white shelf against a wall / lying on a marble surface / held up by a hand]. Photorealistic. High-end publishing quality.
```

## 5.3 YouTube Thumbnail / Social Media Thumbnail

**When to use:** User wants a high-click-through-rate thumbnail image.

```
A [YouTube / podcast / newsletter] thumbnail for the topic: "[TOPIC]".

Style: [bold text-forward / dramatic person-centered / clean minimalist / high-contrast split composition]. Text: "[HEADLINE TEXT]" in [large, bold, high-contrast font / hand-lettered style]. The text [fills one third of the frame / overlaps the subject / sits on a [color] banner].

Visual: [the person from the uploaded image with an expression of [shock / curiosity / excitement / authority] / a dramatic [object / scene] illustration]. Background: [solid [color] / gradient / blurred environment].

Technical: [16:9 landscape, 1280×720 scale]. High-contrast, visually scannable at small size. No small text. [Optional: arrow pointing to subject / reaction face in corner / split-panel before/after].
```

## 5.4 Promotional Poster / Flyer

**When to use:** User wants a designed promotional image for an event, product, or announcement.

```
A [premium / minimalist / bold / vintage] promotional poster for [EVENT / PRODUCT / SERVICE / ANNOUNCEMENT].

Main visual: [hero image description — person, product, scene, abstract element]. Typography hierarchy:
- Headline: "[MAIN TITLE]" — [large, dominant, in X color]
- Subheadline: "[SUBTITLE / TAGLINE]" — [medium size, contrasting color]
- Details: [date, location, URL, price] — [small, clean, readable]

Color palette: [primary color, secondary color, accent color]. Style: [Art Deco geometric / Swiss international / brutalist / hand-made risograph / neon cyberpunk / clean corporate].

Aspect ratio: [portrait A4 / square 1:1 / landscape 16:9]. Print-ready quality. [Include QR code placeholder / No QR code].
```

## 5.5 Chalkboard / Educational Explainer Image

**When to use:** User wants a topic explained visually in a chalkboard teacher style.

```
Using the following content, create a chalkboard-style, hand-written explainer image that breaks down the topic with diagrams and clear, easy-to-understand visuals — as if an enthusiastic teacher wrote it.

Content:
---
[INSERT TOPIC / TEXT / NEWS CONTENT HERE]
---

Style: [green chalkboard with white chalk / white dry-erase board / aged blackboard with colored chalk]. Include: diagrams, arrows connecting concepts, circled key terms, hand-drawn icons illustrating each point, underlines for emphasis. Layout: [structured grid of sections / flowing mind-map / linear top-to-bottom].

Tone: [educational and clear / enthusiastic and playful / technical and precise]. Aspect ratio: [16:9 / 4:3 / 1:1].
```

## 5.6 Header / Banner Image for Articles & Blogs

**When to use:** User wants a visual header for a blog post, newsletter, or note article.

```
Completely recreate the uploaded person [in a stylized illustration / as a photograph]. Make it a header image for a [blog / newsletter / note article / LinkedIn post] where that person introduces "[TOPIC]". Aspect ratio: horizontal 16:9.

Style: [simple, hand-drawn style with a blue and green gradient / clean flat illustration / watercolor / cinematic photograph]. Title text overlaid: "[ARTICLE TITLE]". Typography: [bold sans-serif / elegant serif / handwritten]. Text placement: [left third / bottom third / centered].

The person's likeness should be clearly identifiable. Overall feel: [professional and trustworthy / friendly and approachable / bold and authoritative].
```

## 5.7 Vintage Patent Document / Historical Artifact Style

**When to use:** User wants an image styled as a historical document, certificate, or artifact.

```
A vintage [patent document / certificate / map / wanted poster / newspaper front page] for [SUBJECT — e.g., "a smartphone" / "a coffee maker" / "an AI system"], styled after [late 1800s United States Patent Office filings / 1920s Art Deco certificates / Victorian-era broadsheets].

The page features [precise technical drawings with numbered callouts (Fig. 1, Fig. 2, Fig. 3) showing front, side, and exploded views / ornate border designs and engraved imagery]. [Handwritten annotations / Ornate typography] in [fountain-pen ink / letterpress style] describe [mechanisms / achievements / proclamations]. The paper is aged [ivory / yellowed / sepia] with [foxing stains and soft fold creases / water damage marks]. An official [embossed seal / wax stamp / ribbon seal] appears in the corner. A [hand-signed inventor's name and date / official signature] at the bottom. The entire image feels like a recovered archival document — authoritative, historic, and slightly mysterious.
```
