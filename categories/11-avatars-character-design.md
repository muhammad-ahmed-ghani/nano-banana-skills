# Category 11: Avatars & Character Design

Templates for creating profile pictures, game/trading card characters, 3D avatars, chibi/blind-box figures, caricatures, and character sheets from reference photos or descriptions.

---

## 11.1 3D Blind Box / Pop Mart Style Avatar

**When to use:** User wants to convert a portrait into a cute, collectible 3D toy-figure aesthetic.

```
Transform the person in the uploaded photo into a cute 3D Pop Mart style blind box character.

Likeness: Keep key recognizable features — [hair color, hair style, glasses if any, skin tone, ethnicity, facial structure]. The character must remain clearly the same person translated into chibi form.

Style: C4D rendering, occlusion render, cute Q-version proportions (large head, small rounded body). Soft studio lighting, pastel color palette. Background: a simple, solid matte color (e.g., soft [blue / pink / mint / peach]).

Proportions: Very large head relative to body. Simplified but recognizable facial features. Large, expressive anime-style eyes. Small hands and feet.

Detail: Smooth, plastic toy texture with slight glossy finish. Facing forward with a friendly expression. [Seated in a small display box / Standing on a small base / Floating with decorative elements around it].

Render style: High-quality glossy 3D, similar to a polished Blender render or premium collectible figure. Vibrant but polished colors with strong highlights and soft reflections.
```

## 11.2 3D Chibi Character (Full Anime Style)

**When to use:** User wants a full anime-style chibi render, more detailed than a blind box figure.

```
Transform the subjects into cute anime chibi characters rendered as clean 3D models, while preserving the subject's original race, ethnicity, skin tone, and recognizable facial likeness — including hairstyle, hair color, and clothing colors. The character must remain clearly the same person with the same skin tone and ethnic appearance, translated into chibi form.

Classic chibi proportions: very large head, small torso, simplified cute facial features, large expressive anime-style eyes. Frame from the torso up, camera slightly closer and cropped in.

Camera: slightly dynamic angle — subtle tilt or off-axis perspective, similar to modern anime promotional artwork. Camera slightly lower than the character's face, creating a playful upward angle that makes the character feel lively and engaging.

Render: High-quality glossy 3D anime-style — smooth, reflective, toy-like materials with strong highlights, soft reflections, and shiny surfaces. Bright anime-style shading with vibrant but polished colors.

Background: colorful gradient with a small number of large, translucent decorative shapes — bubbles, stars, or rounded forms — semi-transparent and glossy, softly floating without drawing attention from the character.

Final look: dynamic, glossy 3D chibi character render with polished materials, vibrant colors, and premium collectible aesthetic.
```
