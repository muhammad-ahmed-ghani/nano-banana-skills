# Category 3: Creative & Artistic

Templates for comics, recursive/conceptual art, 3D renders, technical illustrations, and other creative experiments.

---

## 3.1 Multi-Panel Comic / Storyboard

**When to use:** User wants a narrative told across multiple illustrated panels.

```
Create a comic with the title: "[COMIC TITLE]".

Panel 1: [Scene description — setting, characters, action, mood]. [Art style — e.g., "rendered in a poetic watercolor-ink style with fine ink outlines and soft, bleeding washes"]. [Mood/color — e.g., "Muted greys and blues dominate the background. A vivid [color] accent highlights the key element."] [Caption or dialogue].

Panel 2: [Next scene — what changes, what action occurs]. [Same art style consistency]. [Caption or dialogue].

Panel 3: [Scene description]. [Characters' expressions and body language]. [Caption or dialogue].

Panel 4: [Climax or key moment]. [Visual metaphor or symbolic element]. [Caption or dialogue].

Panel 5: [Resolution or final image]. [Overall tone: friendly / dramatic / humorous / inspirational]. Make the aspect ratio [16:9 / 4:3 / 1:1].

Overall style: [watercolor-ink / clean vector / noir graphic novel / manga / European bande dessinée]. Consistent character designs across all panels. [Color palette: limited / full / muted / vibrant].
```

## 3.2 Recursive / Droste Effect Image

**When to use:** User wants an image that contains itself, or an infinite regress visual.

```
[recursive image / Droste effect photograph] of [SUBJECT — e.g., "an orange cat sitting in an office chair holding up an iPad"]. [On the iPad / in the mirror / on the screen / in the painting] is the same [subject] in the same scene [holding up the same iPad / reflecting the same mirror / showing the same canvas]. Repeated [infinitely / 4–5 times] with each iteration progressively smaller and slightly less detailed. The recursion should be clearly visible and mathematically satisfying. [Photography style: photorealistic / illustrated / oil painting / daguerreotype].
```

**Time-anchored variant:**
```
Amateur photograph from [YEAR] of [PERSON DESCRIPTION] [copying / photographing / painting] an image from [a computer screen / a photograph / a canvas], but the image being [copied / photographed / painted] is itself the photo of the [person] [in the act of doing this]. Authentic [era] visual quality — [film grain / digital compression artifacts / sepia tone].
```

## 3.3 Blueprint / Technical Exploded Diagram

**When to use:** User wants an engineering-style technical diagram or exploded-view schematic.

```
A [two-panel / single-panel] technical blueprint diagram in clean monochrome line-art. [LEFT PANEL: a full, intact [OBJECT] in precise thin line-art on a white background.] [RIGHT PANEL / MAIN: a highly detailed, vertically exploded-view diagram showing each component separated into layers: [list key components].]

Use consistent thin [grey / white on dark blue / black] line-weight with no shading. Add numbered circular labels around each part, and include a matching numbered parts list [at the bottom / in the margin]. [Include logo/brand name in the top-left in engineering-manual style.]

The entire artwork should mirror the composition, spacing, typography, and minimalist engineering-manual aesthetic of a professional technical schematic. Blueprint paper color: [white / dark navy / aged cream].
```

## 3.4 Hand-Drawn Isometric Schematic

**When to use:** User wants a hand-sketched isometric view of a location, building, or object.

```
Create a hand-drawn isometric schematic diagram of [SUBJECT — e.g., "Cape Coast Castle, Ghana" / "a Victorian house" / "a Japanese ramen restaurant"]. Style: architectural sketch, pencil or ink on paper, isometric 3D projection. Show [exterior structure / interior cross-section / exploded assembly]. Key details labeled with thin annotation lines and handwritten-style text. [Warm aged paper texture / clean white background]. Hatching used for shadows and depth. Scale bar and compass rose in the corner.
```

**Photo → Schematic:**
```
Create a hand-drawn isometric schematic diagram of the [street / building / location] shown in the uploaded photo. Accurately capture the spatial relationships, key structural features, and proportions. Style: clean architectural sketch with ink outlines.
```

**Schematic → Photo:**
```
Create a photorealistic photograph from this hand-drawn isometric schematic diagram. Render the architecture with accurate materials, realistic lighting [at golden hour / midday / overcast], and environmental context [vegetation, sky, people for scale]. 8K, architectural photography style.
```

## 3.5 Chalkboard / Whiteboard Art Style

**When to use:** User wants content visualized as handwritten on a chalkboard or whiteboard.

```
Using the following content, summarize [the news / the concept / the data / the topic] in a chalkboard-style, hand-written look, and break it down with diagrams and easy-to-understand expressions as if a teacher had written it. Style: [green chalkboard with white chalk / black chalkboard with colored chalk / white dry-erase board with marker]. Include: [diagrams, arrows, underlines, circled key terms, hand-drawn icons]. Tone: [educational / enthusiastic / technical / casual]. Content to visualize:

---
[INSERT CONTENT HERE]
---
```

## 3.6 Polaroid / Scrapbook Storyboard

**When to use:** User wants a nostalgic storyboard or memory collage format.

```
Create an image about "[FILM / NOVEL / STORY / EVENT]" retold through a series of Polaroid photos pinned to a cork board. Each photo captures a key moment, with [handwritten / typewritten] captions below. Arrange the photos in a loosely [chronological / thematic] path across the board, using [colored strings / push pins / washi tape] to connect events and characters. Light the scene warmly to evoke nostalgia. Include incidental details: [coffee cup rings, paper clips, handwritten sticky notes, ticket stubs, dried flowers] for authenticity. [4:3 / 16:9] aspect ratio.
```

## 3.7 Conceptual / Surreal Visualization

**When to use:** User wants an abstract concept or metaphor rendered as a striking image.

**"How X sees Y" format:**
```
[Eiffel Tower / Golden Gate Bridge / [LANDMARK]] as seen through the eyes of [a structural engineer / a child / an alien / a painter]. The visualization reflects their unique mental model: [technical measurements and load diagrams overlaid / childlike proportional distortions and bright colors / incomprehensible geometry / impressionistic brushstrokes]. Cinematic framing, thoughtful composition.
```

**Where's Waldo / "Find the ___" format:**
```
A "Where's [CHARACTER]" scene showing [all [FRANCHISE] characters / a dense crowd of [THEME] figures] in [SETTING — e.g., "Tatooine market / medieval fair / a cyberpunk city street"]. Dense, richly detailed crowd composition. Every character is recognizable but the target is hidden among the chaos. [4:3 landscape / 1:1] format. Illustrated style matching the [franchise / era].
```

## 3.8 Four-Panel Grid Composition with Interaction

**When to use:** User wants a 2×2 or multi-panel grid where all panels connect or interact.

```
High-end photo studio [2×2 grid / 3-panel / triptych] photo.

[PANEL 1 — Top-left / Left / First]: [Background color]: [SUBJECT] wears [OUTFIT DESCRIPTION]. [Action: what they're doing, what they hold, where they look]. [Background elements: text, patterns, objects].

[PANEL 2 — Top-right / Center / Second]: [Background color]: The same [person / character] wears [DIFFERENT OUTFIT]. [Action: how they connect with Panel 1 — puzzle piece meeting, gaze direction, reaching toward].

[PANEL 3 — Bottom-left / Right / Third]: [Background color]: Same [person / character] wears [OUTFIT]. [Action and connection to other panels].

[PANEL 4 — Bottom-right / Final]: [Background color]: The same [person / character]. [Climactic action that completes the grid — puzzle assembled, pattern completed, story resolved]. [Celebratory visual element: confetti burst / golden light / fireworks].

Character consistency: [Maintain precise facial features, retain original face structure across all panels. The character must be completely consistent with the uploaded reference image.]
Lens: 85mm, f/1.8 aperture. Four-panel composition with [puzzle / mirror / narrative] interaction, fashion magazine style.
```
